However, existing approaches typically extract objects from inputs through an arbitrary partitioning of the latent space. We demonstrate that these methods do not respect set invariance and thus have fundamental representational limitations. We propose a novel and general network module that resolves this limitation and demonstrate how it naturally induces object decomposition without explicit supervision.
